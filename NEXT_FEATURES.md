# Next Feature Ideas

1. Add multi-hand support for blackjack games, i.e allow players to play multiple hands simultaneously per round.
2. Add toggle to Recent Sessions section of dashboard to switch to viewing all previous sessions instead of just recent ones, with pagination.
3. In EV/AV chart tab, add toggle to show per session or cumulative EV/AV.
4. Add option to set custom table rules (e.g number of decks, dealer hits/stands on soft 17, double after split allowed, etc) - update session history to reflect these rules.
5. Add counting and perfect basic strategy trainer mode, where users can practice counting cards and making optimal plays without real betting and get real-time feedback on their performance.
6. Add more detailed statistics and analytics, such as win/loss streaks, average bet size over time, and performance against different dealer upcards.
7. Implement multi-player support, allowing users to play blackjack with friends or other online players within the app. This could include chat functionality and leaderboards.
